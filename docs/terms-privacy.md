# Terms of Use / Privacy

SEC Filings QA is provided as-is for convenience. No responsibility is taken for the accuracy, completeness, availability, or suitability of the app, its generated answers, or any source material it retrieves.

AI-generated output can contain mistakes. Always double-check answers against the original SEC filings and any other authoritative sources before relying on them.

This app does not log your questions, filings, API keys, or other usage data to an application server. It runs locally in your browser as a static site. Any data entered into the app remains in your browser except for requests your browser sends directly to the selected AI provider, the SEC, GitHub Pages, or any configured SEC proxy.

The optional browser-session API key storage uses `sessionStorage` on your device. Clearing the browser session removes it.
